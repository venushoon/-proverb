# 🎮 초등 속담/관용구 팡팡! (인터랙티브 웹 게임)

> [!info] 프로젝트 개요
> - **목적:** 초등학생들의 어휘력(속담, 관용구) 향상을 위한 흥미 위주의 상호작용형 웹 게임.
> - **타겟 환경:** 스마트폰, 태블릿, 그리고 교실 내 **전자칠판(멀티 터치 지원)**.
> - **개발 형태:** HTML, CSS, JavaScript 단일 파일 (`.html`). 외부 리소스 최소화.
> - **특징:** 100개의 내장 데이터, 1인용/2인용 경쟁 모드, 오답 노트 및 이미지 캡처 기능 지원.

---

## 🌟 주요 특징 및 기능

### 1. 세 가지 게임 모드
- **📖 학습 모드 (뜻 보고 맞히기):** 속담의 '뜻풀이'를 보고 전체 속담 문장을 객관식으로 고르는 모드.
- **🎮 1인 퀴즈 (빈칸 채우기):** 속담 중간의 빈칸(`[ ]`)에 들어갈 알맞은 단어를 고르는 기본 모드.
- **⚔️ 2인 대결 (동시 터치 지원):** 화면이 좌우로 분할되며, 두 학생이 전자칠판이나 태블릿에서 동시에 터치하며 대결하는 모드. `onpointerdown` 이벤트를 사용하여 멀티 터치 충돌을 방지함.

### 2. 오답 노트 및 피드백 시스템
- **자동 오답 수집:** 1인 퀴즈 및 2인 대결 모드에서 틀린 문제만 모아 결과 화면에서 제공.
- **직관적인 비교:** `나의 오답: ❌ (선택한 단어) ➔ 정답: ⭕️ (진짜 정답)` 형태로 보여주어 메타인지를 돕고 학습 효과를 극대화.
- **이미지 캡처 (스크린샷):** `html2canvas` 라이브러리를 활용하여 완성된 오답 노트를 깔끔한 흰색 배경의 이미지(`png`)로 갤러리에 다운로드하는 기능 지원.
- **오답 복습 게임:** 틀린 문제들만 모아서 다시 풀어볼 수 있는 학습 모드로 바로 연결.

### 3. 교사용 세부 설정 (Settings)
- **주제 선택:** 전체 랜덤 / 일반 속담만 / 동물 관련 속담만 / 관용구만
- **문제 수 조절:** 10문제 / 20문제 / 30문제 / 50문제
- **환경 설정:** 효과음 On/Off, 다크 모드(🌙) 지원.

### 4. 반응형 UI/UX (Fluid Design)
- **CSS Clamp 활용:** 화면 크기(스마트폰~전자칠판)에 따라 글자 크기와 버튼 크기가 유동적으로 변함 (`font-size: clamp(...)`).
- **가독성 최적화:** 모바일 환경에서 버튼 내 텍스트가 잘리지 않도록 줄바꿈(`word-break: keep-all`) 및 패딩 최적화. 뜻풀이가 길어도 생략 없이 모두 표시됨.

## 🛠️ 기술 스택 및 주요 로직

- **DOM 조작:** 바닐라 JavaScript (Vanilla JS)
- **멀티 터치:** `onpointerdown` 이벤트 및 CSS `touch-action: none` 적용.
- **오디오 엔진:** `Web Audio API`를 사용하여 외부 MP3 파일 없이 코드로 직접 정답/오답 효과음(Oscillator) 합성.
- **스크린샷:** `html2canvas.min.js` (CDN 링크 사용)
- **데이터 구조:** Array of Objects 형태.
  ```javascript
  // 데이터 예시
  { 
    t: "animal", // 카테고리
    q: "소 잃고 [ ] 고친다", // 문제 (빈칸)
    a: "외양간", // 정답
    o: ["지붕", "울타리", "대문"], // 오답 보기
    f: "소 잃고 외양간 고친다", // 전체 문장 (학습 모드용)
    m: "일이 이미 잘못된 뒤에는..." // 뜻풀이
  }
